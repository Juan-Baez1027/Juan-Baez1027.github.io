# Personal Portfolio

This is Juan Baez's personal portfolio site, hosted with GitHub Pages at https://juan-baez1027.github.io/. It's a single static page (`index.html` + `styles.css`, no build step) built around an IT support ticket theme. To edit the content — projects, toolkit, about text, or contact links — just edit the text directly inside `index.html`; styling lives in `styles.css`.
