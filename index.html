IT Support Assistant

A desktop chatbot for first-line IT support tickets — account lockouts, VPN, printer issues — built with Tkinter and backed by a local Ollama model, Claude, or DeepSeek. It does real tool use: looks up tickets in SQLite, checks accounts, searches the web, and pulls from an internal knowledge base with semantic (RAG) retrieval and keyword fallback. Diagnostics are opt-in and read-only allowlisted; any remediation is approve-to-run with a full audit trail. API keys live in the OS credential store, and it ships with cross-platform CI on GitHub Actions across Ubuntu, macOS, and Windows on Python 3.10 and 3.12, plus a PyInstaller build to a standalone .exe or .app.

Transit Tracker

A live NYC transit tracker — real-time subway and bus positions on an interactive dark-themed Leaflet map, with per-stop arrival boards showing countdown minutes and each vehicle's destination. Powered directly by the MTA's GTFS-Realtime protobuf feeds, with zero API keys required. A Flask backend exposes a JSON REST API and caches feed pulls for about 30 seconds to stay within polling limits, while the frontend is plain HTML and JavaScript with no build tooling. Covers all 345 searchable bus routes and 496 subway stations.
